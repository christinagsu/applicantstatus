eAcceptance Documentation

Generate Letters

  1. Click the "Home" tab
  2. Choose the Student Type, Offer Code, Application Term, Date Range, and Admission Type
  3. Click Go
  4. If you are happy with the template, click Send. If you would like to change the template, click Edit.

Change Residency Paragraphs

  1. Click the "Residency Paragraphs" tab
  2. Choose the Residency Type you would like to edit and click Choose.
  3. Edit the text and click Submit.

View Previous Mailings

  1. Click the "Previous Mailings" tab.
  2. Filter using one of the two filters at the top of the page.
  3. Click on the eAcceptance letter you would like to view.
